# Commands to push fixes to GitHub

# 1. Check current status
git status

# 2. Create and switch to new branch
git checkout -b fix/cors-parser-sandbox

# 3. Add all changes
git add .

# 4. Commit with descriptive message
git commit -m "fix: CORS, tree-sitter parser, and sandbox execution issues

- Fix CORS errors
  - Updated CORS middleware with explicit origins
  - Added expose_headers for better compatibility
  - Support both localhost and 127.0.0.1

- Fix tree-sitter parser
  - Updated Language initialization for new tree-sitter API
  - Use set_language() method instead of passing to Parser
  - Added proper error handling

- Fix sandbox execution on Windows
  - Switched from asyncio.create_subprocess_exec to subprocess.run
  - Resolves NotImplementedError on Windows
  - Better error messages and logging
  - More lenient security patterns (allow basic file operations)

- Improve error handling and logging across services"

# 5. Push to GitHub
git push -u origin fix/cors-parser-sandbox

# 6. After pushing, create a Pull Request on GitHub
# Go to: https://github.com/YOUR_USERNAME/YOUR_REPO/pull/new/fix/cors-parser-sandbox
